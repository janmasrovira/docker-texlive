* Readme
Custom image of the texlive 2026 distribution.

It contains most of the packages (~4k) but only supports English. The
=collection-fontsrecommended= set is included; no other extra fonts are
installed.

** Building locally and pushing
#+begin_example
cd docker-texlive
docker buildx build . --tag test
-- run the image (optional)
docker run -it --rm --name testrun test
-- tag with final name
docker tag test janmasrovira/texlive:2026-custom
-- push
docker push janmasrovira/texlive:2026-custom
#+end_example
